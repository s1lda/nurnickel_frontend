## Как запустить frontend часть?
* Перейдите в директорию frontend
```commandline
cd frontend
```
* Создайте .env файл и ничего в нём не меняйте
```commandline
cp .env.example .env
```
* Установите все зависимости
```commandline
npm install 
```
* Запустите скрипт следующей командой:
```commandline
npm run dev
```
